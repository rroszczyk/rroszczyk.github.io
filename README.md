## Krótka instrukcja

budowa całości:
```bash

JEKYLL_ENV=production jekyll build

```

wersja lokalna do edycji:

```bash

jekyll serve --livereload

```

### Jekyll Themes

Your Pages site will use the layout and styles from the Jekyll theme you have selected in your [repository settings](https://github.com/rroszczyk/roszczyk.github.io/settings). The name of this theme is saved in the Jekyll `_config.yml` configuration file.
